# FUEGO ENTERTAINMENT

Sacramento's premier nightlife events — Archive Fridays, Traviesa Saturdays, Deseo Sundays.

## Repository Structure

```
fuegoent-frontend/   Next.js 16 + Tailwind v4 + shadcn/ui website
fuegoent-backend/    Express API (coming soon)
```

## Frontend

```bash
cd fuegoent-frontend
npm install
npm run dev          # http://localhost:3000
npm run build
```

## Backend

```bash
cd fuegoent-backend
npm install
npm run dev          # http://localhost:5000
```
